SCP 命令

发表于 2019-01-17 |

SCP

SCP 是 Secure Copy 的简写，用于 Linux 下进行远程拷贝文件的命令， SCP 传输是经过加密的。

SCP 用途

1.获取远程服务器上的文件

2.将本机上的文件上传到远程服务器上

远程服务器既没有配置 ftp 服务器，没有开启 web 服务器，也没有做共享，无法通过常规途径获取/上传文件时，只需要通过 scp 命令便可轻松的达到目的。
